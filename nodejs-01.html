Node.js的用途

  1. 基于Express框架构建Web应用
  2. 基于Electron框架构建跨平台的桌面应用
  3. 基于restify框架构建API接口项目

Node.js的学习路径

Javascript基础语法 + 内置API接口fs、http + 第三方框架

终端常用操作: 上箭头 历史命令 TAB 补全路径 ESC 清除命令 cls 清空终端

fs文件系统模块

  1. fs.readFile(path[,option],callback) 读取文件内容
  2. fs.writefile(file,data[,options],callback) 写入文件内容 data 为写入数据 []为可选参数,会覆盖掉原本的内容

path路径模块

  1. path.join(path,path,path,) 将多个路径模块拼接成一个完整的路径
  2. path.basename(path[,'.扩展名']) 获取路径的最后一个部分，也可以去除扩展名
  3. path.extname() 获取路径中的扩展名

HTTP模块: 创建Web服务器

  1. IP地址
  2. 域名与域名服务器
  3. 端口号 对应web服务；一个端口号对应一个web服务；在实际应用中，URL中端口80可以被省略；
  4. req 请求对象 包含与客户端有关的数据和属性：url地址 method 请求类型
  5. res 响应对象

模块化基本概念

  1. 模块来源不同：内置模块 自定义模块 第三方模块
  2. 模块作用域 内部定义变量不会被外部使用
  3. 通过module.exports向外共享属性和方法

npm与包

  1. npm install 包的名字
  2. const 包名 = require('包名')
  3. 2.24.0 第一位数字代表大版本 第二位数字代表功能版本 第三位数字代表BUG修复版本
